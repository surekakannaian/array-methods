ARRAY METHODS

1.concat()

The concat() method concatenates (joins) two or more arrays:

const arr1 = ["Cecilie", "Lone"];
const arr2 = ["Emil", "Tobias", "Linus"];
const arr3 = arr1.concat(arr2);

2.copyWithin(target,start,end)

copyWithin() copies array elements to another position in an array, overwriting existing values:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

3.entries()

entries() returns an Array Iterator object with key/value pairs:

const fruits = ["Banana", "Orange", "Apple", "Mango"];
const f = fruits.entries();

4.every(funcion(current value,index,arr))

every() returns true if all elements in an array pass a test (provided as a function).

const ages = [32, 33, 16, 40];

6.filter(funcion(current value,index,arr))

The filter() method creates a new array filled with elements that pass a test provided by a function.

Get every element in the array that has a value of 18 or more:

const ages = [32, 33, 16, 40];

7.find(funcion(current value,index,arr))

The find() method returns the value of the first element that passes a test.

Find the value of the first element with a value above 18:

const ages = [3, 19, 18, 20];

8.findIndex(funcion(current value,index,arr))

The find() method returns the index of the first element that passes a test.

Find the value of the first element with a value above 18:

const ages = [3, 19, 18, 20];

9.forEach(funcion(current value,index,arr))

The forEach() method calls a function for each element in an array.

Multiply the value of each element with 10:

const ages = [3, 19, 18, 20];

10.Array.from(object)

The Array.from() method returns an array from any iterable object.

Create an array from a string:

let text = "ABCDEFG";
const myArr = Array.from(text);

11.includes(element)

The includes() method returns true if an array contains a specified value.
The includes() method returns false if the value is not found.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

12.indexOf(item,start)

The indexOf() method returns the first index (position) of a specified.
The indexOf() method returns -1 if the value is not found.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

13.isArray(obj)

The isArray() method returns true if an object is an array, otherwise false.

let text = "W3Schools";

14.join(seperator)

The join() method returns an array as a string.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

15.keys()

keys() returns an Array Iterator object with the keys of an array:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

16.lastIndexOf()

he lastIndexOf() method returns the last index (position) of a specified value.
The lastIndexOf() method returns -1 if the value is not found.

const fruits = ["Orange","Apple","Mango","Apple","Banana","Apple"];
let index = fruits.lastIndexOf("Apple");

17.length

The length property sets or returns the number of elements in an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

18.map()

map() creates a new array from calling a function for every array element.
map() calls a function once for each element in an array.

const numbers = [4, 9, 16, 25];

19.pop()

The pop() method removes (pops) the last element of an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

20.push()

The push() method adds new items to the end of an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

21.reverse()

The reverse() method reverses the order of the elements in an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

22.shift()

The shift() method removes the first item of an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

23.slice()

Array.slice() returns selected array elements as a new array:

const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];

24.sort()

The sort() sorts the elements of an array.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

25.splice()

The splice() method adds and/or removes array elements.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.splice(2, 0, "Lemon", "Kiwi");

const fruits = ["Banana", "Orange", "Apple", "Mango", "Kiwi"];

fruits.splice(2, 2);

26.toString()

The toString() method returns a string with array values separated by commas.

const fruits = ["Banana", "Orange", "Apple", "Mango"];

27.unshift()

unshift() adds new items to the beginning of an array:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

fruits.unshift("Lemon", "Pineapple");

29.valueOf()

valueOf() returns the array itself:

const fruits = ["Banana", "Orange", "Apple", "Mango"];

const myArray = fruits.valueOf();
